Getting Started

CodingBat is a free site of live Java and Python coding problems to build coding skill. Each problem has a problem description and a table showing some sample output for that problem.

Type your code into the large text area and click the "Run Tests" button to save your code, compile and run. Each time you click "Run Tests", the results are shown in the bottom of the page.

Coding Help

See the Java Help and Python Help pages for many resources and videos on various coding topics. This is a great place to see examples and get help if you are stuck on a problem.

Coding Ground Rules

Your code should just be a simple method or function with the prototype just as it is in the starter code. The code should compute and return the value (don't print the result value).

Additional Notes:
  • If the method takes a String or array argument, those arguments will never be passed in as null. However, the empty String or array is still a valid case unless the problem statement specifically says otherwise.
  • You may write helper methods, although most often, the problem can be solved in with code short enough to fit in one method.
  • In Java, the import for the java.util classes is already done if you would like to use those classes. Python code should not print and should not do any imports.

Running Green

When your code compiles and runs correctly, you get a green check for that problem. If the output of a run says "Timed out", your code probably includes an infinite loop or other bug that prevents it from finishing.

There may be a few "Hidden Tests" listed at the bottom of the table that are not detailed. These test cases are similar to the ones shown in the table, but with different inputs that are not visible to you. These test cases test the comprehensiveness of your code and make sure it runs for a variety of test cases, not just the ones listed.

Saving Work with Accounts

If you create an account, you can come back to the server another time and your code, and you checkmarks, etc. will be saved. You can do everything on the server without an account, but your work will be forgotten when your session ends after 2 hours without activity.
